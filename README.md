# Multi-Linear Regression Model

This repository contains a lab notebook for building and analyzing a Multi-Linear Regression Model.

## Lab File

- [Lab03_Multi_LinearRegression_Model.ipynb](https://github.com/joccy-code/Multi-LinearRegression--Model/blob/main/Lab03_Multi_LinearRegression_Model.ipynb)

## Project Overview

The lab notebook demonstrates the application of multi-linear regression using Python and common data science libraries. It guides users through data preparation, model building, evaluation, and visualization techniques.

## Features

- Exploratory Data Analysis (EDA)
- Preparing Data for Regression
- Constructing a Multi-Linear Regression Model
- Evaluating Model Performance (e.g., R-squared, RMSE)
- Visualizing Results

## Requirements

- Python (Recommended: 3.7 or later)
- Jupyter Notebook
- Libraries:
  - pandas
  - numpy
  - scikit-learn
  - matplotlib

You can install requirements via pip:

```bash
pip install pandas numpy scikit-learn matplotlib
```

## Usage

1. Clone the repository:
    ```bash
    git clone https://github.com/joccy-code/Multi-LinearRegression--Model.git
    ```
2. Open the lab notebook in Jupyter:
    ```bash
    jupyter notebook Lab03_Multi_LinearRegression_Model.ipynb
    ```
3. Follow the step-by-step guidance in the notebook to build your own multi-linear regression model.
